# Food App - Backend

Domain based GraphQL API for the food app

## Available Scripts

- yarn test: runs unitary tests
- dev: start development mode
- lint: check code styleguide
- lint:format: apply code styleguide rules

## Contributing

- Please follow http://karma-runner.github.io/5.2/dev/git-commit-msg.html for commit guidelines

## How to install

Create a .env file following the env example

run `yarn` to install dependencies

## How to run

Just run `yarn dev` to boots up the development mode
